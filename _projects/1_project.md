The goal of the `examplercts` package is to make it easy to access documented,
analysis-ready data from randomized trials across various therapeutic areas.
Examples include trials with different outcome types, including continuous,
binary, ordinal, and time-to-event data. Each dataset includes a data
dictionary, and are annotated to indicate which variables are assessed prior
to randomization.

Example data is critical to teaching and providing realistic worked examples of
applying methods and software.

The `examplercts` package pulls data from the following packages:

  - [`survival` package](https://cran.r-project.org/web/packages/survival/index.html)
  - [`medicaldata` package](https://higgi13425.github.io/medicaldata/)
  
Note that `medicaldata` contains many other example datasets from observational
studies and some smaller randomized trials.
